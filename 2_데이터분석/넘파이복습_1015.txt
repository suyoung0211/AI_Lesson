넘파이 행렬 (일반 리스트가 아니라 ndarray 타입 배열 객체)
용어 : 스칼라,벡터,행렬,텐서 

1. 기본 산술 연산 :  +,-,*,/ , //, %
2. 1차원 행렬 내적 : 각 원소를 곱하고 결과를 더하기(벡터의 크기)

↪ 1,2번은 행렬의 shape 이 같아야 합니다.

3. 2차원 이상 다차원 행렬의 곱 :
   ↪ 2차원 행렬의 곱은 자주 사용됩니다.
      A,B 피연산 행렬의 내적을 구해서 결과 행렬이 만들어집니다.
   ↪ m x n 행렬,  p x q 행렬의 n =p 같아서 1차원 행렬의 내적을
    구할 수 있어야 합니다. 결과는 m x q 행렬 . (내적의 갯수)

4. shape 의 변경
   1,2,3차원 기존 행렬을 각각 다른 차원으로 변경
   ↪ reshape

5. 축(axis) 의 변경
   ↪ 2,3차원 행렬의 대해 축을 서로 바꾸기.(값의 방향이 바뀜)
   ↪ shape 이 transpose 가 되었을 때 , 변경 결과는 예측을 해야합니다.
   ↪ (z,y,x) 축인 3차원을 axis 값 0,1,2  으로 하여 다시 나열하면 됩니다.
      : 인자가 없으면 기본값 2,1,0  즉 (x,y,z) 로 변경

6. 위의 함수(또는 연산) 중에서 축을 지정하는 것
   : 산술연산
   : np.sort() 

7. 넘파이 배열에서 range 처럼 값의 범위로 1차원 배열 만드는 함수
   : np.arange(start,stop, step)   -> stop 포함하지 않음
   : np.linspace(start,end, 갯수)    -> end 포함
